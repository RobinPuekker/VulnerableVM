# VulnerableVM
Vulnerable VM — Project Exam 2
A deliberately vulnerable Ubuntu Server 22.04 virtual machine implementing four web application vulnerabilities from the 2025 MITRE CWE Top 25 list. Built for cybersecurity education and penetration testing practice.
Vulnerabilities

SQL Injection (CWE-89)
Cross-Site Scripting (CWE-79)
OS Command Injection (CWE-78)
Unrestricted File Upload (CWE-434)
